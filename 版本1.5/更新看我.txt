admin/index.php	        後台總覽儀表板（統計卡片、排行榜）
admin/players.php	玩家管理（搜尋、停權、重置、詳細資料）

admin/db_layer.php	資料庫層：連線狀態、資料表總覽、怪物生成預覽、傷害模擬器、升級閾值表
admin/api_module.php	API 模組：訓練/戰鬥 API 監控統計、即時記錄、API 測試工具
admin/login.php	        管理員登入頁（獨立）
admin/auth.php	        後台 Session 驗證守衛
